# Cumulus Labs — Pricing (2026 年版)

## 価格表

| Plan | 月額 (per seat) | 主な制限 |
| ---- | -------------- | -------- |
| Starter | $49 | 5 seats まで / dashboard 5 個 / API rate 10 req/sec |
| Pro | $199 | 50 seats まで / dashboard 50 個 / API rate 100 req/sec / SSO 含む |
| Enterprise | 個別見積 | seat 無制限 / dashboard 無制限 / 専任 CSM / 専用 region |

## 年間契約割引

- 年間前払いで 20% off (Starter / Pro 両方)
- Enterprise は契約期間と utilization 約束で個別 negotiate

## 料金変更

- 半期に 1 回まで (1 月 / 7 月) 改定可能。直前 60 日に既存顧客へ通知必須
- 既存顧客は現契約期間中は旧料金で固定

## サポート

- Starter: メールサポート、初動 2 営業日
- Pro: メール + Slack connect、初動 1 営業日
- Enterprise: メール + Slack + 電話、初動 4 時間 (Sev1 は 1 時間)
