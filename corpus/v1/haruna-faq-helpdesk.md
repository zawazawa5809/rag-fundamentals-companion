# ハルナ・テクノロジーズ — 社内ヘルプデスク FAQ

> 最終更新: 2026 年 4 月 18 日 (社内 IT サポート)

過去 6 ヶ月に Slack `#help-it` に寄せられた質問のうち、頻度の高い 12 件をまとめた。

## 労務

**Q: 在宅勤務手当はいくらですか？**
A: 月額 5,000 円が自動付与される。詳細は `haruna-remote-work.md`。

**Q: 有給休暇は何日もらえますか？**
A: 入社日から 10 日付与、6 ヶ月経過時にさらに付与。年次更新は 4 月 1 日。

**Q: 育休制度はありますか？**
A: 法定 + 自社制度で最大 2 年間取得可能。配偶者出産休暇は 5 日 (有給)。詳細は人事部に問い合わせ。

## 経費

**Q: 領収書の原本はいつまでに送ればいいですか？**
A: 5,000 円以上は原本必須。社内便で経理部へ。詳細は `haruna-expense.md`。

**Q: 経費精算の締切は？**
A: 当月分は翌月 5 日まで。3 月 (決算月) のみ別途案内。

**Q: PR ってなんですか？**
A: ハルナでは **2 つの意味で使われる**。経費精算文脈では「Purchase Request = 立替・精算」の略、購買部門では「事前購買リクエスト」の略。文書中の文脈で判断する。

## IT

**Q: VPN に接続できません。**
A: まず Yubikey が刺さっているか確認、次に Tailscale を再起動。それでも駄目なら `#help-it` へ。

**Q: Slack のチャンネル作成権限は？**
A: 全社員に付与済み。`#proj-` プレフィクスは admin 承認、それ以外は自由。

**Q: Pegasus の API キーを再発行したい。**
A: SRE 部の `#sre-pegasus` で「API key rotation」と書いてリクエスト。即日対応。

## オンコール

**Q: オンコール手当はいくら？**
A: 通常給与 + 週次 30,000 円 (税込)。ページング発生時の実工数は別途残業計上。

**Q: 翌月曜の flex day って？**
A: オンコール直後の月曜は出社不要の有給扱い。事前申請不要。

**Q: PagerDuty の通知が来ない時は？**
A: モバイル app の通知許可とサイレント設定を確認。次に Slack `#oncall-engineering` へ post。
