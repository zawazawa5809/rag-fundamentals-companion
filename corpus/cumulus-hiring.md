# Cumulus Labs — Hiring プロセス

## 4 段階フロー

### Stage 1: Recruiter screen (30 分)

- Recruiter による文化的フィットと motivation の確認
- 給与 range / リモート可否 / start date の合意
- 通過: 80% 前後

### Stage 2: Hiring Manager interview (60 分)

- 該当 squad の lead が担当
- 直近の業務 deep dive + Cumulus でやりたいこと
- 通過: 50% 前後

### Stage 3: Technical / Craft loop (180 分、複数日に分割可)

- Engineer ロール: design exercise + pair programming + code review
- Designer ロール: portfolio walk + critique session + practical task
- PM ロール: roadmap critique + stakeholder roleplay
- 評価者: 3 名 (squad lead + cross-functional 2 名)
- 通過: 30% 前後

### Stage 4: Founder chat (45 分)

- CEO もしくは CTO
- Mission alignment + 長期視点
- offer 出すかどうかの最終判断
- 通過: 90% 前後

## オファー

- offer 提示は Stage 4 から 5 営業日以内
- equity の説明は別 doc で詳細送付
- 入社日: offer 受諾後 2-12 週

## Bar

- 「Cumulus にいなかった場合、その squad の output は本当に下がるか」を全員に問う
- ボーダー判定は不採用に倒す (false positive を避ける)

## 多様性

- 各 stage で評価者の 2 名以上は候補者と異なる demographic background から構成
- panel debrief で「unconscious bias の可能性」を必ず議論
